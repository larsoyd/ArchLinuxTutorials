# Complete Arch Linux Tutorial w/ Automounting Partitions & KDE Plasma 6

This is a OPINIONATED Arch installation guide for people that just want a working system to game on & daily drive. This tutorial is optimized for KDE & Wayland, but a SonicDE (X11/XLibre) path is provided for those who would rather continue to use KDE but with X11.
Tutorial is split into a Install tutorial which is the ArchISO/chroot part of the tutorial, while the Post-Install is after rebooting into the system. Post-Install tutorial is required but not explicitly necessary. 

- A bootable Arch Linux USB (written with `dd` or similar)
- Internet connection
- UEFI system
