# Complete Arch Linux Tutorial w/ Automounting Partitions

This is a OPINIONATED Arch installation guide for people that just want a working system to game on. This tutorial is optimized for KDE & Wayland, but a SonicDE (X11/XLibre) path is provided for those who would rather continue to use KDE but with X11.

- A bootable Arch Linux USB (written with `dd` or similar)
- Internet connection
- UEFI system
